# 2048.w30

A [2048](https://github.com/gabrielecirulli/2048) game port for web30.

## Building

First install the dependecies needed by the build script:

```
pip3 install -r ./requirements.txt
```

Then run the script:

```
python3 ./build.py
```

The output is saved to `dist/` folder
