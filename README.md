# 2048.xdc

A [2048](https://github.com/gabrielecirulli/2048) game port for webXdc.

## Test

To test the app you can simply open the `index.html` file.

## Build

```sh
pip3 install -r ./requirements.txt
python3 ./build.py
```

The output is a file with `.xdc` extension.
